
Smart-Refrigerator - v2 yolov5
==============================

This dataset was exported via roboflow.ai on December 30, 2020 at 1:26 PM GMT

It includes 1010 images.
Food-items are annotated in YOLO v5 PyTorch format.

The following pre-processing was applied to each image:
* Resize to 416x416 (Stretch)

The following augmentation was applied to create 3 versions of each source image:
* 50% probability of horizontal flip
* Random rotation of between -15 and +15 degrees
* Random brigthness adjustment of between -12 and +12 percent
